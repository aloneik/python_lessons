Системы счисления
    
1. Напишите класс для перевода из 10-тичной системы счисления в римскую
2. Добавьте класс, который будет расширять функционал предыдущего, добавляя возможность обратной конвертации
3. Добавьте еще один класс на основе предыдущего (расширить функциональность) с возможностью двусторонней конвертации из/в двоичную, восьмиричную и шестнадцатиричную системы.
4. Создайте класс, который будет позволять вызывать экземпляры своих объектов как функции.
Например: a = ExecutableClass()
		  a()  # Do something

Сделайте этот метод, который вызывается когда вызываем класс (a()) абстрактным и определите для каждого из предыдущих классов, предварительно отнаследовав их от этого класса.
Реализация метода должна делать то, что делает класс.
Например, для класса обратной конвертации из римской в десятичную, этот метод должен принимать аргументы - число и куда конвертировать и производить конвертацию
5. Для каждого класса опеделеите возможность работы операций сложения, вычитания умножения и деления для чисел в тех системах счисления, с которыми этот класс работает.
Например: класс из пункта 2 должен иметь возможность делать вышеуказанные операции для чисел в римской системе и в десятичной. 
Класс из пункта 3 помимо этого дожен работать еще и с двоичной, восьмиричной и шестнадцатиричной системаи.
Другими словами, если передать экземпляр класса с числом X(10 - в римской системе) и другой объект с число 8 (в десятичной) и сложить их, то получим 18, если первым будет 8 или XVIII, если первое число X
6. Добавьте возможность сравнения чисел
