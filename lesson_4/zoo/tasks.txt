Интерактивный зоопарк 

1. Создайте абстрактный класс животного и определите в нем 5 поля: имя, тип, спецификация, масса, изображение, голос. 
Также добавьте 3 метода: получение имени, изображения и голоса
Метод получения изображения должен работать показать изображение этого животного. При этом поле изображения содержит имя и путь к файлу изображения.
Пример:
>>> from PIL import Image                                                                                
>>> img = Image.open('test.png')
>>> img.show() 
Тоже самое для метода голоса.
Пример:
>>> import winsound
>>> winsound.PlaySound('cat.wav', winsound.SND_FILENAME)
2. Создайте 3 класса каждого типа животного: птица, млекопитающее, рептилия.
При этом каждый класс должен иметь свои специальные поля:
Млекопитающее - LitterSize: среднее количество потомства, которое есть у этого млекопитающего (>=1)
Рептилия - VenomousOrNot: указывает, имеет ли эта рептилия ядовитый укус
Птица - Wingspan: размах крылье в сантиметрах (>=1)
		TalksOrMute: разговаривает или нет
		Для тех птиц, которые разговаривают, есть еще одно поле: Phrase - фраза, которую птица говорит
3. Проверьте, что будет выведено в результате выполнения print(a), где a - экземпляр какого-либо класса.
Сделайте так, чтобы эта инструкция выводила данные об экземпляре в строку в виде "Имя Тип Спецификация Масса (<Значние дополнительных параметров>, Изображение, Звук)"
Например: Bob Mammal Bear 300 (2, bob_bear.png, bob_bear.waw)
4. Создайте класс зоопарка, объекты которого будут содержать объекты зверей.
5. Добавьте возможность добавления зверей вручную, через командную строку.
Например: 
Animal Name? Bob
Animal Type? Mammal
Animal Species? Bear
Animal Litter Size? 2
Animal Image? bob_bear.png
Animal Sound? bob_bear.waw
6. Добавьте возможность добавления зверей через файл
7. Реализуйте возможность поиска по имени
8. Реализуйте возможность определения аттрибута.
Например:
Query animal species[s], mass[m], litter[l], venom[v], wingspan[w], talk[t] or exit session[e]? s
Animal Name? Bob
Bob species is Bear

Query animal species[s], mass[m], litter[l], venom[v], wingspan[w], talk[t] or exit session[e]? m
Animal Name? Lucy
Lucy mass is 2

Query animal species[s], mass[m], litter[l], venom[v], wingspan[w], talk[t] or exit session[e]? t
Animal Name? Polly
Polly talks.
Polly says I want a cracker
9. Добавить возможность сохранения данных в файл
10. Добавить возможность быстрого вывода данных зоопрака построчно путем вызова интсрукции print
11. Добавить возможность итерирования по объекту зоопарка

P.S. Для работы с файлами просьба использовать модуль pickle.
