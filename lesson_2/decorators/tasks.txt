Упражнения по декораторам:
1) Есть функция get_time которая возвращает список из 3 строк: дата и время (часы и минуты). 
Нужно задекорировать оригинальную функцию так чтобы результатом была строка вида 
“It is {date}. Since {hour} hours and {minutes} minutes.”.
Реализация функции get_time тоже должна быть представлена
2) Модифицируйте декоратор из предыдущего шага, чтобы он принимал аргументы.
Задекорированная функция должна возврашать строку вида - “Now it is {date}”
Модифицируйте функцию get_time - должна принимать аргумент, в котором будет указан 
формат даты. Значение по умолчанию для этого аргумента будет "%Y/%m/%d %H:%M".
Функция get_time должна возвращать строку с датой в указанном формате.
3) Есть список мобильных номеров (должны быть введены вручную). 
Нужно отсортировать их в порядке возрастания и вывести 
в следующем ввиде:
+91 ххххх ххххх
Введенные мобильные номера могут начинаться с +91, 91 или 0 перед 10-ти значным номер. 
Или вообще могут быть указаны без префикса.
Пример ввода:
07895462130
919875641230
9195969878
Результат
+91 78954 62130
+91 91959 69878
+91 98756 41230
Подсказка - функция должна парсить и сортировать номера, декоратор - выводить на экран в
указанном формате
4) Напишите декоратор, который позволяет сохранить результат функции (кэшировать), 
чтобы не вычислять результат функции повторно.
Если результата функции нет в кэше, то выполнить вычисления и положить результат в кэш.
